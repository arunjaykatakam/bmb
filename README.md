# BorrowMyBricks Shopify Theme

This is a lightweight Shopify Online Store 2.0 theme scaffold rebuilt from the BorrowMyBricks Gamma landing page.

## Run with Shopify CLI

```bash
shopify theme dev
```

## Homepage structure

- Hero
- How it works
- Subscription plans
- Postage and returns
- FAQ
- CTA

All homepage sections are editable from the Shopify theme customizer.
